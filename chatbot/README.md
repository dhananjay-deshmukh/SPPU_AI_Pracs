# A medical expert system.

## Run Locally

Clone the project

```bash
  git clone https://https://github.com/dhananjay-deshmukh/SPPU_AI_Pracs.git
```

Go to the project directory

```bash
  cd chatbot
```

Install streamlit

```bash
  pip install streamlit
```

Run the app

```bash
  streamlit run chatbot.py
```
